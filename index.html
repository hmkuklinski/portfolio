About Me

I've always had a love for learning foreign languages. I started learning Korean by myself because I enjoyed watching Korean drama and variety shows. My goal initially was to learn the language so I could understand the lyrics to my favorite songs and be able to read the subtitles. I never expected that this hobby would lead me to learn about computer languages and software development.


I decided I would take an introduction to programming with Java class over the summer to see if computer science was something that would interest me. I loved learning about how programs were structured and also the endless possibilities of applications in the real world. After this class, I started learning HTML, CSS, JavaScript, and Python. I love making projects because they allow me to apply what I've learned and express my creativity. It also allows me to improve on my skills! It gives me the ability to see how much I've grown as a developer between edits.
